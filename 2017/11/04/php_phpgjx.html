mysql追踪调试神器phpgjx2.0工具箱

2017-11-04 php php 阅读0次 评论

此工具是tpshop上学习时候看到的非常实用，只要有数据库访问就可以显示所有sql语句

下载工具phpgjx2.0

  • 修改 C:\Windows\System32\drivers\etc\hosts 文件 比如加入 127.0.0.1 www.sql.com

  • 修改 apache 配置文件 D:\xampp\apache\conf\extra\httpd-vhosts.conf文件

  • 修改 phpgjx2文件夹中config.php 文件 配置mysql连接 重启MYSQL

  • 修改D:\xampp\mysql\bin\my.ini文件
