sublime text 3作为markdown编辑器实时预览

2017-11-03 sublime text markdown 阅读0次 评论

第一步安装插件

  • 安装MarkDown Editing（编辑和语法高亮支持 ）和Markdown Preview插件（导出html预览支持 ）
  • ctrl+shift+p, 然后输入 Install Package
  • 输入Markdown Editing, 回车
  • 注：如果你安装Markdown Editing完之后，遇到了如下的错误，那么你安装的时候可能开着一个Markdown文件，所以卸载完之后在不打开Markdown的情况下再次安装就可以解决了。

  • 同样安装Markdown Preview, 回车

第二步配置插件

  • Preferences → Package Settings → Markdown Preview → Setting 打开后将左边default的设置代码复制到右边User栏，之后找到”enable_autoreload”: flase,把flase改成true（这部分很重要），重启sublime text3.

第三步使用

  • 新建一个md文件
  • 按下ctrl+shift+P打开快速菜单，键入’mp’

  • 现在可以在浏览器看到，md每修改一次保存，刷新游览器就可以及时看到了。
